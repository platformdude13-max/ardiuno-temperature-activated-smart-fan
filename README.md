# Temperature Activated Smart Fan

A smart fan system that automatically turns on or adjusts its speed based on the surrounding temperature. 
---

## Features

- Monitors temperature using a sensor that detects it pretty fast.
- Automatically turns the fan ON/OFF or adjusts speed based on temperature thresholds.
- Real-time temperature display via LCD or serial monitor.

---

## Technologies & Components

### Hardware
- Microcontroller (e.g., Arduino Uno / ESP32 / Raspberry Pi)
- Temperature Sensor (DHT11 / DHT22 / LM35)
- DC Fan or PWM-controlled fan
- Transistor or Relay Module (to drive fan)
- Resistors, Breadboard, Jumper wires

### Software
- Arduino IDE / PlatformIO
- C / C++ (for Arduino)
